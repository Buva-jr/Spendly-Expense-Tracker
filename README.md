# Spendly - Personal Finance Tracker

A full stack personal finance management application.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Context API, Axios
- **Backend:** Spring Boot, Spring Security, Spring Data JPA
- **Database:** MySQL

## Features
- Add income and expense transactions
- Dashboard with real-time balance, income, expense summary
- Transaction history with category filters
- Budget tracking with progress bars
- Secure REST API with Spring Security
